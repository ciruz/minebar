# Security

## Reporting a Vulnerability

If you find a security issue, please don't open a public GitHub issue.

Email: hi@minebar.app

I'll respond within 48 hours and aim to fix confirmed issues within 7 days.

## Scope

MineBar communicates only with devices on your local network. No data is sent to external servers.

## Out of Scope

- Issues requiring physical access to your network
- Vulnerabilities in third-party mining firmware (AxeOS, Avalon firmware)
